# Website for PTSD Treatment

This project is a prototype telehealth application designed to support PTSD treatment through digital tools.

## Features

- User Authentication (Firebase)
- Dashboard Interface
- Mood Tracker
- Mood History Visualization (Graphs)
- Optional Notes for Daily Mood Entries

## Technologies Used

- React.js
- Firebase Authentication
- Firebase Firestore
- Chart.js
- JavaScript / HTML / CSS
